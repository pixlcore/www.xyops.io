# xyOps.io Website Repo

This repository holds all the files for the `xyOps.io` website.

It is designed as a [PoolNoodle App](https://github.com/jhuckaby/PoolNoodle#app-installer) and can be installed using the `noodle` CLI:

```sh
noodle install github:pixlcore/www.xyops.io#v1.0.0
```
